# อาหารไทยและความรู้ทั่วไป

## ต้มยำกุ้ง
ต้มยำกุ้งเป็นอาหารไทยที่มีชื่อเสียงระดับโลก มีรสชาติเผ็ด เปรี้ยว เค็ม และหอมกลิ่นสมุนไพรไทย

วัตถุดิบหลัก:
- กุ้งสด (กุ้งแม่น้ำหรือกุ้งทะเล)
- น้ำซุปหรือน้ำเปล่า
- ข่า ตะไคร้ ใบมะกรูด
- พริกขี้หนูสด
- น้ำปลา
- น้ำมะนาว
- เห็ดฟาง หรือเห็ดนางฟ้า
- หอมแดง
- ผักชี
- น้ำพริกเผา (สำหรับต้มยำน้ำข้น)
- นมข้น หรือกะทิ (สำหรับต้มยำน้ำข้น)

วิธีทำต้มยำกุ้ง:
1. ต้มน้ำให้เดือด ใส่ข่า ตะไคร้ และใบมะกรูดลงไปเพื่อให้กลิ่นหอม
2. ใส่กุ้งลงไปต้มจนสุก
3. ปรุงรสด้วยน้ำปลา น้ำมะนาว 
4. หากทำต้มยำน้ำข้น ให้ใส่น้ำพริกเผาและนมข้นหรือกะทิ
5. โรยด้วยผักชีและพริกขี้หนูสด

ประวัติ:
ต้มยำกุ้งมีต้นกำเนิดในประเทศไทย และได้รับความนิยมแพร่หลายไปทั่วโลก ในปี พ.ศ. 2554 ต้มยำกุ้งได้รับการจัดอันดับเป็นอาหารอันดับที่ 8 ที่อร่อยที่สุดในโลกโดยนิตยสาร CNN Travel

## ผัดไทย
ผัดไทยเป็นอาหารจานเดียวที่มีชื่อเสียงของไทย ประกอบด้วยเส้นจันท์ผัดกับไข่และเครื่องปรุงรส เสิร์ฟพร้อมถั่วงอก ใบกุยช่าย และเครื่องเคียงอื่นๆ

วัตถุดิบ:
- เส้นจันท์ (เส้นผัดไทย)
- ไข่ไก่
- กุ้งแห้ง
- เต้าหู้เหลือง
- ถั่วงอก
- ใบกุยช่าย
- น้ำตาลปี๊บ
- น้ำมะขามเปียก
- น้ำปลา
- พริกป่น
- ถั่วลิสงบด
- มะนาว

วิธีทำผัดไทย:
1. แช่เส้นจันท์ในน้ำจนนิ่ม
2. ทำน้ำปรุงผัดไทยโดยผสมน้ำตาลปี๊บ น้ำมะขามเปียก และน้ำปลา
3. ผัดเต้าหู้และกุ้งแห้งให้หอม ตอกไข่ลงไปผัดให้สุก
4. ใส่เส้นจันท์ที่แช่น้ำแล้วลงไปผัดให้เข้ากัน
5. เติมน้ำปรุงลงไปผัดให้เส้นดูดซับน้ำปรุง
6. ใส่ถั่วงอกและใบกุยช่ายลงไปผัดแค่พอเข้ากัน
7. เสิร์ฟพร้อมถั่วลิสงบด พริกป่น และมะนาว

ประวัติ:
ผัดไทยเริ่มเป็นที่นิยมในประเทศไทยในช่วงสงครามโลกครั้งที่ 2 เนื่องจากรัฐบาลไทยส่งเสริมให้คนไทยบริโภคเส้นก๋วยเตี๋ยวแทนข้าว เพื่อลดการบริโภคข้าวซึ่งเป็นสินค้าส่งออกที่สำคัญในขณะนั้น ชื่อ "ผัดไทย" มีความหมายว่า "ผัดแบบไทย" ซึ่งแสดงถึงความเป็นชาตินิยมในช่วงนั้น

## แกงเขียวหวาน
แกงเขียวหวานเป็นแกงกะทิรสเผ็ดของไทย มีสีเขียวจากพริกเขียว มักใส่เนื้อไก่ เนื้อวัว หรือกุ้ง

วัตถุดิบ:
- เนื้อไก่ (หรือเนื้อวัว หรือกุ้ง)
- น้ำกะทิ
- พริกแกงเขียวหวาน (พริกเขียว หอมแดง ข่า ตะไคร้ ผิวมะกรูด กระเทียม รากผักชี กะปิ)
- มะเขือเปราะ
- มะเขือพวง
- ใบโหระพา
- ใบมะกรูดฉีก
- น้ำปลา
- น้ำตาลปี๊บ

วิธีทำแกงเขียวหวาน:
1. ผัดพริกแกงเขียวหวานกับหัวกะทิจนแตกมัน
2. ใส่เนื้อสัตว์ลงไปผัดให้สุก
3. เติมน้ำกะทิลงไป ต้มให้เดือด
4. ใส่มะเขือเปราะ มะเขือพวง ปรุงรสด้วยน้ำปลา น้ำตาลปี๊บ
5. เมื่อผักสุก ใส่ใบโหระพาและใบมะกรูดฉีก ปิดไฟ

ประวัติ:
แกงเขียวหวานมีต้นกำเนิดในภาคกลางของไทย โดยเชื่อว่าเริ่มแพร่หลายในสมัยรัชกาลที่ 2 แห่งกรุงรัตนโกสินทร์ คำว่า "หวาน" ในชื่อไม่ได้หมายถึงรสชาติหวาน แต่หมายถึงสีเขียวอ่อนที่สดใสของแกง

## ส้มตำไทย
ส้มตำไทยเป็นอาหารประเภทยำที่มีรสชาติเปรี้ยว เผ็ด หวาน เค็ม กลมกล่อม

วัตถุดิบ:
- มะละกอดิบ
- มะเขือเทศ
- ถั่วฝักยาว
- กระเทียม
- พริกขี้หนู
- น้ำตาลปี๊บ
- น้ำมะนาว
- น้ำปลา
- กุ้งแห้ง
- ถั่วลิสง

วิธีทำส้มตำไทย:
1. โขลกกระเทียมและพริกให้ละเอียด
2. ใส่มะเขือเทศและถั่วฝักยาวลงไปตำพอแตก
3. ใส่มะละกอที่ซอยเป็นเส้นลงไปตำให้เข้ากัน
4. ปรุงรสด้วยน้ำตาลปี๊บ น้ำมะนาว น้ำปลา
5. ใส่กุ้งแห้งลงไปคลุกให้เข้ากัน
6. โรยด้วยถั่วลิสง

ประวัติ:
ส้มตำมีต้นกำเนิดจากอาหารของชาวลาวที่อาศัยอยู่ในภาคตะวันออกเฉียงเหนือของไทย (อีสาน) ต่อมาได้แพร่หลายไปทั่วประเทศและมีการปรับรสชาติให้เข้ากับคนไทยทั่วไป โดยรสชาติจะไม่เผ็ดจัดเท่าส้มตำลาว และเพิ่มความหวานมากขึ้น

## ต้มข่าไก่
ต้มข่าไก่เป็นซุปกะทิที่มีรสชาติเปรี้ยว เผ็ด และหอมกลิ่นเครื่องสมุนไพร โดยเฉพาะข่าซึ่งเป็นส่วนประกอบสำคัญ

วัตถุดิบ:
- เนื้อไก่
- น้ำกะทิ
- น้ำซุป
- ข่าอ่อน
- ตะไคร้
- ใบมะกรูด
- พริกขี้หนูสด
- เห็ดฟางหรือเห็ดนางฟ้า
- น้ำปลา
- น้ำมะนาว
- ผักชีหรือใบสะระแหน่

วิธีทำต้มข่าไก่:
1. ต้มน้ำซุปกับกะทิให้เดือด
2. ใส่ข่า ตะไคร้ และใบมะกรูดลงไปต้ม
3. ใส่เนื้อไก่ลงไปต้มจนสุก
4. ใส่เห็ดลงไป ปรุงรสด้วยน้ำปลา น้ำมะนาว
5. โรยด้วยพริกขี้หนูซอยและผักชีหรือใบสะระแหน่

ประวัติ:
ต้มข่าไก่เป็นอาหารที่มีต้นกำเนิดในภาคกลางของไทย เป็นที่นิยมมาตั้งแต่สมัยโบราณ เนื่องจากข่าเป็นพืชที่ปลูกง่ายและมีสรรพคุณทางยา ช่วยแก้อาการไอ และบำรุงธาตุ

## ข้าวหมกไก่
ข้าวหมกไก่เป็นอาหารไทยที่ได้รับอิทธิพลมาจากอาหารมุสลิม เป็นข้าวหุงกับเครื่องเทศหอมและกะทิ เสิร์ฟกับไก่ที่ปรุงรสเครื่องเทศ

วัตถุดิบ:
- ข้าวหอมมะลิ
- เนื้อไก่
- กะทิ
- หอมแดง กระเทียม
- รากผักชี
- ขมิ้น
- อบเชย
- กานพลู
- ลูกผักชี
- ใบกระวาน
- พริกไทยดำ
- น้ำตาล น้ำปลา
- อาจิโนะโมะโต๊ะ (ผงชูรส)

วิธีทำข้าวหมกไก่:
1. หมักไก่ด้วยเครื่องเทศบดละเอียด
2. หุงข้าวกับกะทิและเครื่องเทศ
3. นำไก่ที่หมักไว้ไปทอดหรืออบจนสุก
4. เสิร์ฟข้าวหมกพร้อมกับไก่ และอาจมีเครื่องเคียงเช่น แตงกวา น้ำจิ้ม และไข่ต้ม

ประวัติ:
ข้าวหมกไก่เป็นอาหารที่ได้รับอิทธิพลมาจากอาหารเปอร์เซียและอาหารอินเดีย ผ่านทางชาวมุสลิมที่อพยพมาอาศัยในประเทศไทย โดยเฉพาะในภาคใต้และกรุงเทพฯ แพร่หลายในประเทศไทยในสมัยอยุธยา

## ผัดกระเพราหมูสับ
ผัดกระเพราหมูสับเป็นอาหารจานด่วนยอดนิยมของไทย มีรสชาติเผ็ด เค็ม และหอมใบกระเพรา มักเสิร์ฟกับข้าวสวยร้อนๆ และไข่ดาว

วัตถุดิบ:
- หมูสับ (หรือเนื้อสัตว์อื่นๆ เช่น ไก่ เนื้อ ทะเล)
- ใบกระเพรา
- พริกขี้หนู
- กระเทียม
- น้ำมันพืช
- น้ำปลา
- น้ำตาล
- ซอสหอยนางรม
- ซีอิ๊วดำ (ให้มีสีสวย)

วิธีทำผัดกระเพราหมูสับ:
1. โขลกหรือสับพริกและกระเทียมให้ละเอียด
2. ตั้งกระทะใส่น้ำมัน ใส่พริกกระเทียมที่โขลกไว้ลงไปผัดจนหอม
3. ใส่หมูสับลงไปผัดให้สุก
4. ปรุงรสด้วยน้ำปลา น้ำตาล ซอสหอยนางรม และซีอิ๊วดำ
5. ใส่ใบกระเพราลงไปผัดให้เข้ากันแค่พอใบกระเพราสลด
6. เสิร์ฟร้อนๆ กับข้าวสวย นิยมทานคู่กับไข่ดาว

ประวัติ:
ผัดกระเพราเป็นอาหารที่คิดค้นขึ้นในประเทศไทยไม่นานมานี้ ประมาณ 50-60 ปีที่ผ่านมา โดยใช้ใบกระเพรา ซึ่งเป็นสมุนไพรที่มีกลิ่นหอมเฉพาะตัวและมีสรรพคุณทางยา เช่น ช่วยขับลม แก้ท้องอืด เป็นส่วนประกอบหลัก กลายเป็นอาหารจานด่วนยอดนิยมเพราะทำง่าย รสชาติจัดจ้าน และอิ่มท้อง

## มัสมั่น
มัสมั่นเป็นแกงกะทิรสจัดของไทยที่ได้รับอิทธิพลจากอาหารมุสลิม มีรสชาติหวาน เผ็ด และมีกลิ่นหอมของเครื่องเทศ

วัตถุดิบ:
- เนื้อวัวหรือไก่
- มันฝรั่ง
- หอมหัวใหญ่
- ถั่วลิสง
- กะทิ
- พริกแกงมัสมั่น
- น้ำตาลปี๊บ
- น้ำปลา
- น้ำมะขามเปียก
- อบเชย
- กานพลู
- ใบกระวาน
- ลูกจันทน์เทศ

วิธีทำแกงมัสมั่น:
1. ผัดพริกแกงมัสมั่นกับหัวกะทิจนแตกมัน
2. ใส่เนื้อสัตว์ลงไปผัดจนสุกดี
3. เติมกะทิและน้ำซุป ใส่เครื่องเทศลงไป
4. ใส่มันฝรั่งและหอมหัวใหญ่ ต้มจนนิ่ม
5. ปรุงรสด้วยน้ำตาลปี๊บ น้ำปลา น้ำมะขามเปียก
6. ใส่ถั่วลิสงลงไป ต้มต่อจนน้ำแกงข้น

ประวัติ:
มัสมั่นเป็นแกงที่ได้รับอิทธิพลมาจากอาหารเปอร์เซียและอินเดีย ผ่านทางชาวมุสลิมที่เข้ามาในประเทศไทยตั้งแต่สมัยอยุธยา คำว่า "มัสมั่น" มีที่มาจากคำว่า "มุสลิม" แกงมัสมั่นปรากฏในวรรณกรรมไทยเก่าแก่ เช่น กาพย์เห่ชมเครื่องคาวหวาน สมัยรัชกาลที่ 2

## ขนมจีนน้ำยา
ขนมจีนน้ำยาเป็นอาหารไทยที่ประกอบด้วยเส้นขนมจีน (เส้นหมักทำจากแป้งข้าวเจ้า) ราดด้วยน้ำยาที่ทำจากปลาหรือกะทิ เสิร์ฟพร้อมผักสดหลากหลายชนิด

ประเภทของขนมจีนน้ำยา:
1. น้ำยาป่า - ทำจากปลาย่าง กะปิ และสมุนไพร รสจัด
2. น้ำยากะทิ - ทำจากปลาและกะทิ รสนุ่ม หวานกว่าน้ำยาป่า
3. น้ำเงี้ยว - เป็นอาหารไทยภาคเหนือ ทำจากเนื้อหมูและมะเขือเทศ
4. น้ำพริก - เป็นน้ำพริกข้นๆ ราดบนเส้นขนมจีน

วัตถุดิบน้ำยาป่า:
- ปลาช่อนหรือปลาทู
- กระชาย
- พริกแห้ง
- ข่า ตะไคร้
- ใบมะกรูด
- กระเทียม หอมแดง
- กะปิ
- น้ำปลา
- มะนาว

วัตถุดิบน้ำยากะทิ:
- ปลาช่อนหรือปลาทู
- กะทิ
- พริกแกงเผ็ด
- น้ำปลา
- น้ำตาล
- มะนาว

ผักเครื่องเคียง:
- ถั่วงอก
- ผักกาดดอง
- ใบกะเพรา โหระพา แมงลัก
- ผักบุ้ง
- ถั่วพู
- มะเขือเปราะดิบ แตงกวา
- มะนาว พริกขี้หนูซอย

ประวัติ:
ขนมจีนมีต้นกำเนิดจากอาหารจีนที่เข้ามาสู่ประเทศไทยตั้งแต่โบราณ แต่คนไทยได้ดัดแปลงวิธีการทำและรสชาติให้เข้ากับความชอบของคนไทย โดยใช้แป้งข้าวเจ้าแทนแป้งสาลี และพัฒนาน้ำยาหลากหลายรูปแบบ

## ข้าวผัด
ข้าวผัดเป็นอาหารจานเดียวยอดนิยมของไทย ทำจากข้าวสวยผัดกับไข่และเครื่องปรุงรส อาจใส่เนื้อสัตว์และผักตามชอบ

วัตถุดิบพื้นฐาน:
- ข้าวสวยหุงสุก (ควรเป็นข้าวค้างคืน)
- ไข่ไก่
- กระเทียมสับ
- หอมใหญ่หรือหอมแดง
- ต้นหอม ผักชี
- น้ำมันพืช
- น้ำปลา
- ซีอิ๊วขาว
- น้ำตาล
- พริกไทย

เนื้อสัตว์ (เลือกตามชอบ):
- หมู ไก่ เนื้อ
- กุ้ง ปลาหมึก ปู
- ปลาเค็ม
- แหนม

ผัก (เลือกตามชอบ):
- ข้าวโพดอ่อน
- แครอท
- ถั่วลันเตา
- มะเขือเทศ

วิธีทำข้าวผัด:
1. ผัดกระเทียมในน้ำมันให้หอม
2. ใส่เนื้อสัตว์ลงไปผัดจนสุก
3. ใส่ไข่ลงไปผัดให้สุก
4. ใส่ข้าวสวยลงไป ผัดให้เข้ากัน
5. ปรุงรสด้วยน้ำปลา น้ำตาล ซีอิ๊วขาว พริกไทย
6. ใส่ผักลงไปผัดให้สุก
7. โรยด้วยต้นหอมซอยและผักชี
8. เสิร์ฟพร้อมแตงกวา มะนาว และพริกน้ำปลา

ประวัติ:
ข้าวผัดเป็นอาหารที่ได้รับอิทธิพลจากอาหารจีน แต่คนไทยได้ดัดแปลงให้มีรสชาติถูกปากคนไทยมากขึ้น โดยเพิ่มน้ำปลาและเครื่องปรุงไทย ข้าวผัดในแต่ละภูมิภาคของไทยมีลักษณะเฉพาะแตกต่างกันไป เช่น ข้าวผัดภาคใต้อาจมีการใส่กะปิเพิ่มความหอม
